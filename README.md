# Swasthya.AI

A comprehensive healthcare management platform connecting Patients, Doctors, and Administrators.

## Features
- **Patient:** Book appointments, view medical records.
- **Doctor:** View appointments, access patient records via ID.
- **Admin:** Manage users, view statistics.

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Backend Setup
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
